# photo-player
Photo and video slideshow player for Android
# 相册播放器

一个简单的Android相册播放器应用，支持图片和视频循环播放。

## 功能特性

- 📸 支持图片循环播放
- 🎥 支持2分钟以内的视频播放
- ⚙️ 可自定义图片显示时长（3-30秒）
- 🔄 支持顺序播放
- 📱 横屏全屏显示

## 支持的格式

**图片：** JPG, JPEG, PNG, GIF, WEBP, BMP

**视频：** MP4, AVI, MOV, MKV, 3GP

## 下载安装

从 [Releases](https://github.com/你的用户名/photo-player/releases) 页面下载最新的APK文件。

## 使用方法

1. 打开应用后，授予存储权限
2. 点击"选择文件夹"按钮
3. 选择包含照片/视频的文件夹（如 DCIM/Camera）
4. 应用会自动开始播放

## 构建

本项目使用GitHub Actions自动构建APK。

每次提交代码后，会自动生成新的APK文件。
